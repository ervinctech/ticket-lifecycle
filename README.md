# osTicket - Ticket Lifecycle Examples

## Overview

This project demonstrates the complete lifecycle of support tickets within osTicket. The lab follows tickets from creation through assignment, troubleshooting, escalation, resolution, and closure while simulating real-world IT support scenarios.

## Technologies Used

- Microsoft Azure
- osTicket
- Remote Desktop Protocol (RDP)

## Operating Systems Used

- Windows 10 Pro (21H2)

## Skills Demonstrated

- Help Desk Operations
- Incident Management
- Ticket Escalation
- Customer Support
- Troubleshooting
- Documentation
- Service Desk Workflow

## Lab Objectives

- Create Support Tickets
- Assign Tickets
- Escalate Tickets
- Resolve Issues
- Document Actions
- Close Tickets
